# 📊 Sales Data Cleaning & Delivery Analysis

## 📌 Project Overview
This project focuses on cleaning and analyzing a real-world sales dataset using Python and Pandas.

---

## 🧹 Data Cleaning
- Handled missing values in Postal Code  
- Converted data types (dates and postal codes)  
- Fixed inconsistent formats  
- Created a new feature: Delivery Time  

---

## 📈 Key Insights

### 🚚 Shipping Performance
- First Class delivery: ~2.17 days  
- Standard Class delivery: ~5 days  
👉 First Class is ~2.5x faster  

---

### 🌍 Regional Performance
- Delivery time is consistent (~4 days)  
- Central region is slightly slower (~4.07 days)  
- East region is fastest (~3.91 days)
